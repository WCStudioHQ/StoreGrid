A lightweight Full Site Editing (FSE) theme built for grid-based store layouts. WCS StoreGrid provides a clean and minimal foundation perfect for portfolios, stores (WooCommerce-ready in future versions), or business websites using the WordPress block editor.

== Description ==

**WCS StoreGrid** is a modern, block-based WordPress theme designed for Full Site Editing. It provides a structured, responsive grid layout ideal for showcasing content in a stylish and accessible way. Whether you're building a product showcase, blog, or business landing page, WCS StoreGrid offers clean typography, customizable color palettes, and a modular design that works beautifully across devices.

**Features:**
- WordPress Full Site Editing support (FSE)
- Clean and responsive grid layout
- Customizable header, footer, and templates via site editor
- Global styles support (color, typography)
- Lightweight and fast
- Minimal design with accessibility in mind

== Installation ==

1. Upload the theme to your `/wp-content/themes/` directory or install it via the Appearance > Themes panel.
2. Activate the theme.
3. Visit Appearance > Editor (Site Editor) to begin customizing your site with blocks and templates.

== Frequently Asked Questions ==

= Does WCS StoreGrid support Full Site Editing? =

Yes, WCS StoreGrid supports the WordPress Site Editor, allowing you to fully customize templates and template parts.

= Can I create a child theme for WCS StoreGrid? =

Yes, you can create a child theme for WCS StoreGrid if you need to customize functionality or styles beyond what the block editor offers.
